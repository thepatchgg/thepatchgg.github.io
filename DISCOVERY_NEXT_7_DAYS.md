# Discovery Next 7 Days

This is the short execution list for The Patch while Sugarfest is still recent and Search Console data is still maturing.

## Goal

The site does **not** have a product-quality problem or a retention problem right now.

The main goal for the next week is to improve **discoverability** by:

1. routing more visitors into the site's highest-performing pages
2. keeping live-update content highly visible while it is still relevant
3. watching for the first useful Search Console query and landing-page patterns

Watch these first in GA and Search Console:

## This Week's Moves

### 1. Keep the live Sugarfest guide featured

- Keep the Candy Egg spend guide visible on the homepage and Adopt Me hub.
- If another Sugarfest-adjacent update lands, publish the follow-up guide the same day.
- Link every new Sugarfest/update article to:
  - Trade Calculator
  - Egg Guide
  - Pet Value List
  - Pet Encyclopedia

### 2. Treat the calculator and egg guide as traffic hubs

- The Trade Calculator is the clearest conversion page once someone wants to act.
- The Egg Guide is one of the highest-engagement pages and should keep feeding:
  - Egg EV Tool
  - Trade Calculator
  - Pet Value List
  - Live event guides when relevant

### 3. Review internal traffic filtering

- Check whether internal/dev traffic is filtered out in GA.
- If not, fix that before drawing strong conclusions from small user counts.
- Be cautious with "views per user" and "average engagement" until internal traffic is known to be clean.

### 4. Request indexing for fresh pages

In Search Console, request indexing for:

1. `/articles/adopt-me-sugarfest-candy-egg-spend-guide.html`
2. any new pet pages published this week
3. any newly added comparison or decision guides

## Search Console Review Loop

Run this once or twice this week as data appears.

### Pages and signals to look for

- High impressions + low CTR:
  rewrite title/meta first
- Good engagement + low impressions:
  improve homepage/hub/internal links and request indexing
- New page with impressions:
  create adjacent pages in the same cluster
- Update guide gets clicks:
  add stronger links into calculator, egg guide, and value list

### Questions to answer

1. Which page is getting impressions first?
2. Which page is getting clicks first?
3. Are visitors landing on tools, articles, or pet pages?
4. Does the Candy Egg spend guide start ranking while Sugarfest is still fresh?

## Content Triggers

If an official Adopt Me update post lands:

1. publish the update guide the same day
2. add the tile to the homepage and Adopt Me hub
3. link it into the relevant tool page
4. request indexing in Search Console

If Search Console starts showing egg-specific impressions:

1. expand adjacent egg pages
2. review egg page titles and meta descriptions
3. add stronger egg-to-tool internal links

If pet pages start getting impressions:

1. build more pages around the same pet tier or event family
2. add cross-links between the pet page, value list, and calculator

## For Claude / Other Agents

Use this file as the short-term growth playbook.

Important rules:

- Do not invent traffic wins that are not visible in GA/Search Console.
- Do not recommend broad generic SEO work if page-level opportunities are already clear.
- Prefer:
  1. better internal linking
  2. stronger titles/meta on pages with impressions
  3. more timely update/event coverage
  4. more pages around clusters already showing demand
- Treat the Trade Calculator, Egg Guide, Pet Encyclopedia, Pet Value List, and the latest live update guide as the current priority surfaces.
