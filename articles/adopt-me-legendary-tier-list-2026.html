How to use this board

Think of this as a quick read on legendary benchmarks, not a full replacement for the value list. Demand and movement still matter, and some legendary pets are not in the benchmark set yet.

Use it for direction

Use this page to understand which legendary pets currently anchor the market and which ones are better stepping-stones than long-term goals.

Use the value list for precision

The value list is still the better page when you need exact benchmark numbers, variants, or confidence context.

Use the trade calculator for deals

Once you know the lane a pet belongs to, move to the calculator to test the actual trade instead of trusting memory.

What is not on this board yet

Legendary pets outside the benchmark dataset are intentionally excluded instead of being ranked with low-confidence filler values.

What gets a pet moved up

Consistent value, strong demand, liquidity, and scarcity all matter. A pet with stable movement is often more useful than a “cool” pet with weak demand.

This benchmark board is a directional tool. The shared value list remains the source of truth for exact supported values.
